# 鸭腿阿姨

一个微信聊天框形式的虚构经营小游戏。玩家以“鸭腿阿姨”的第一视角经营校园烤腿生意，在群聊、供应商、家人、舆论和监管消息之间做选择。

## 运行

这是静态页面项目，直接打开 `index.html` 即可运行。

也可以用任意静态服务器部署仓库根目录，例如 GitHub Pages、Cloudflare Pages、Vercel 或 Netlify。

## 文件

- `index.html`：页面结构
- `styles.css`：界面样式
- `game.js`：剧情、状态和交互逻辑
- `assets/duck-leg-aunt-avatar.webp`：玩家头像

## 说明

本游戏为虚构游戏。游戏中的人物、学校、群聊、店铺、事件与经营细节均为虚构，如有巧合，纯属雷同。
